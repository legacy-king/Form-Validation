# Form Validation Project

A practice project for learning client-side form validation using vanilla JavaScript, built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## âœ¨ Features

- **Live validation feedback** - Fields are validated when the user leaves them (on blur)

- **Visual indicators** - Green borders for valid fields, red borders for invalid fields
- **Email format validation** - Uses regex to ensure proper email structure
- **Password strength requirements** - Minimum 8 characters
- **Password confirmation matching** - Ensures passwords match before submission
- **Success feedback** - High five message when form is successfully submitted
